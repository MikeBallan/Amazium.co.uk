Offsetting

Amazium also has the option to offset a column, which should make life all the bit easier if you you need to add an extra bit of space between columns. This is done by adding the class offset followed by how many columns you want the offset to be i.e. if you want to have an offset of 4 then a column of 8 all you need to do is add offset_4 after grid_8.

4
3 with 2 offset
3 with offset 1
4 with 1 offset
7 with 2 offset
Code
<div class="row">
    <div class="grid_4">...</div>
    <div class="grid_4 offset_4">...</div>
</div>

Media queries

Media queries are what makes it possible to change our websites to fit on all the different devices we use day to day and as there are so mny differnt screen sizes to display for, we have setup Amazium to have to have 4 set Media queries which are:

Code
/***** Screen 1200px+ *****/
All Default CSS is put at the top.

/***** Screen Smaller than 1200px *****/
@media only screen and (min-width: 960px) and (max-width: 1199px) {...}

/***** Tablet (Smaller than 959px) *****/
@media only screen and (min-width: 768px) and (max-width: 959px) {...}

/***** Mobile *****/
/***** Updated based on essencious Mobile version *****/
@media only screen and (max-width: 767px) {...}

Now I have to say I 'm a little biased towards most things Apple which means the sizes I've chosen works perfectly for iPad & imobile and everything else I've been able to test Amazium on seems to work fine also..!
