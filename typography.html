Typography

Amazium comes with all the styling you need for H1 to H6 with the default font family set to Sans-Serif which gives a nice contrast to the body copy which is set at 15px Sans-Serif with a line height of 20px. As well as sorting out the body copy weve also created a class for an intro paragrahp which is set to have a font-size & line height of 130% the base font size which make helps if you dont want to use an H2 and strong isnt enough.

Hi, I'm an intro paragraph..!

Code
<p class="intro">...</div>

Button's

Buttons are aways needed on a website and so we have styled some buttons for you to use from the start by simply adding class="btn" to your link. As well as this you can string a color to the btn class class="btn red".

Default link Button Red link Button Blue link Button Green link Button White link Button

If you are in need of a smaller button then all you need to do is add the class "small" to your link string and you will get smaller buttons.

Default Small link Button Red Small link Button Blue Small link Button Green Small link Button White Small link Button

Code
<a href="#" class="btn">...</a>
<a href="#" class="btn red">...</a>
<a href="#" class="btn small">...</a>

And finally if you need to use a button the same code as above applys too..!

Code
<button class="btn">...</button>
<button class="btn red">...</button>
<button class="btn small">...</button>

Blockquote

So for a long time i've been using the cite in my Blockquote as the author of the quote but doing some reasearch into the correct use i found out that the cite should be the url sorce which should be in the blockquote as below.

Code
<figure>
    <blockquote cite="http://www.amazium.co.uk">
        <p>I don't want to sound like a badass but... I eject my usb without removing it safely..!</p>
    </blockquote>
    <figcaption>
        Mike Ballan
    </figcaption>
</figure>

Tables

Also included are styling for tables which to be honest I haven't used that much outside building emails but it's there in case you ever need it..!

Item: Category: Size:
Bananas Fruit 1KG
Apples Fruit 3KG
Oranges Fruit 2KG
Code
<table summary="This is an example of a table">
    <thead>
        <tr>
            <th scope="col">Item:</th>
            <th scope="col">Category:</th>
            <th scope="col">Size:</th>
        </tr>
    </thead>
    <tbody>
        <tr class="even">
            <td>Bananas</td>
            <td>Fruit</td>

            <td>Apples</td>

Images & Media

Seeing how you website will be responsive there will be times when you will need your images to be responsive to and with this in mind I've made a nice little style to make any image 100% responsive, all you need to do is add this little class to your image: class="img-max" and hay presto responsive images..!

Code
<img src="img/blank.jpg" border="0" alt="blank image" class="img-max">

The images sbove are sitting in a grid_5 a& grid_5 but you'll notice that the width & height changes based on the screen size.

Media

As mentioned above about images I suspect that you would also like to have all your videos be responsive as well and with this in mind I had to do a little searching around on the web to find the answer for this which is very simple.

All you will need to do is wrap your video with a div that has a class called .media and thats it one responsive video..!

Vimeo

YouTube

Code
<div class="media">
    <iframe src="video_link_here" frameborder="0"></iframe>
</div>
