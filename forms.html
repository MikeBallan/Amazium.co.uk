Forms

Now forms can be one of the biggest pains in the neck for web developers, so I have included a simple form in Amazium with some very basic styling which has all the elements you will need to make your own form.

So below os a basic form with labels which we have styled all the normal form elements you would expect to see in a for including Select, Radio & Checkboxes which will invove you adding some extra markup.

Select's

We all know select inputs look rather rubish so we have done our best to make it looks the same as all the other imputs in Amazium, in doing this you will need to wrap your select in a span with a class of "select" to make the magin happen.

Invalid entry
Invalid entry
Code
<fieldset>
    <label for="options">Options</label>
    <span class="select">
        <select id="dropdown" name="dropdown" tabindex="18" />
            <option selected>Option 1</option>
            <option>Option 2</option>
            <option>Option 3</option>
        </select>
    </span>
    <small>Invalid entry</small>
</fieldset>

Now you see me, Now you don't lables

Thanks to the awesome Shahrukh Omar who has made it possible to show the label of an input when the user starts typing in a field so know what each field is. All you will need to do is add class="nysmnyd-labels" to your form and make sure you have the Amazium.js file included and you too will have fancy show/hide lables on your inputs.
